# Getting started

## base-shell

Creating a **shell** starter kit is a fairly straight forward process:

```
npx create-react-app my-app --template base
```

## mui-shell



```text
npx create-react-app my-app --template material-ui
```

## rmw-shell

```text
npx create-react-app my-app --template rmw
```
